# Prints

Esta pasta reúne capturas de tela do protótipo e do processo do NEXO.

## Regras

- Incluir apenas prints sem dados pessoais ou sensíveis.
- Não publicar prints com CPF, RG, telefone, assinatura, arquivos reais de PDV ou pessoas identificáveis.

Sugestões de prints: telas do fluxo ADMIN e CLIENTE do protótipo, dashboard de indicadores e organização do repositório/quadro de tarefas.
