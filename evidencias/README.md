# Evidências — NEXO (Projeto Integrador I)

Esta pasta reúne evidências do processo de desenvolvimento que possam ser publicadas com segurança (sem dados sensíveis).

## O que deve entrar aqui

- Prints de telas do protótipo e do quadro de tarefas (GitHub Projects/Trello), sem dados pessoais.
- Links relativos para artefatos já versionados no repositório.
- Registros de reuniões e cerimônias, quando não contiverem dados sensíveis.

## Situação atual

- Evidências consolidadas: **Pendente de inclusão pela equipe**.
- O registro de links da entrega está em [`docs/00_entrega_final/links_entrega.md`](../docs/00_entrega_final/links_entrega.md).

## Regra de dados sensíveis

Não publicar aqui: CPF, RG, telefone pessoal, assinaturas, termo de anuência assinado ou prints com pessoas identificáveis. Esses materiais devem ser enviados apenas pelo canal oficial da disciplina (Sala Online/AVA), se solicitados.
