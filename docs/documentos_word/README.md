# Documentos Oficiais em Word - NEXO PI I

Esta pasta deve concentrar as versões oficiais em Word dos documentos da entrega do **Projeto Integrador I - Análise e Desenvolvimento de Sistemas**, quando a publicação desses arquivos for permitida.

Se os arquivos `.docx` oficiais não estiverem versionados neste repositório, eles devem ser enviados pelo canal oficial da disciplina, como AVA/Sala Online, conforme orientação da professora.

## Cuidados com documentos sensíveis

Não devem ser publicados neste repositório documentos sensíveis, incluindo:

- termo de anuência assinado;
- CPF, RG, telefone pessoal ou assinatura;
- prints com pessoas identificáveis;
- dados reais de PDV ou informações comerciais sensíveis.
