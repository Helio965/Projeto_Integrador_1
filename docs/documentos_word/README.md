# Documentos Oficiais em Word - NEXO PI I

Esta pasta deve concentrar as versoes oficiais em Word dos documentos da entrega do **Projeto Integrador I - Analise e Desenvolvimento de Sistemas**.

Se os arquivos `.docx` oficiais nao estiverem versionados neste repositorio, eles devem ser enviados pelo canal oficial da disciplina, como AVA/Sala Online, conforme orientacao da professora.

## Cuidados com documentos sensiveis

Nao devem ser publicados neste repositorio documentos sensiveis, incluindo:

- termo de anuencia assinado;
- CPF, RG, telefone pessoal ou assinatura;
- prints com pessoas identificaveis;
- dados reais de PDV ou informacoes comerciais sensiveis.
