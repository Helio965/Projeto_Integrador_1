# Relatório Reflexivo / Extensão - NEXO

## Aprendizados da equipe

O desenvolvimento do NEXO reforçou a importância de partir de um problema real antes de propor telas ou funcionalidades. O uso de Design Thinking ajudou a organizar dores, persona e jornada do usuário, enquanto o backlog permitiu transformar a proposta em histórias e tarefas acompanháveis.

Também ficou evidente a importância de delimitar o escopo do Projeto Integrador I como protótipo acadêmico/MVP, evitando prometer funcionalidades de sistema completo antes de validação e implementação real.

## O que funcionou

- Organização do repositório por etapas do ciclo do PI I.
- Definição de problema, público-alvo, requisitos e backlog.
- Construção de um protótipo navegável para demonstrar o fluxo principal.
- Separação entre documentos já estruturados e itens que dependem de execução real.

## O que precisa melhorar

- Registrar dados reais de teste de usabilidade.
- Completar evidências permitidas da execução das sprints e validações.
- Evoluir o protótipo para implementação técnica em etapas futuras, caso o projeto avance.
- Consolidar links externos apenas quando realmente existirem.

## Relação com extensão / comunidade

O projeto se conecta a pequenos comércios que já possuem dados em sistemas de PDV, mas nem sempre conseguem interpretar essas informações para apoiar decisões comerciais. A proposta do NEXO busca tornar esses dados mais compreensíveis por meio de indicadores gerenciais, dashboard e devolutiva estratégica.

## Contribuição do projeto

Como protótipo acadêmico, o NEXO contribui ao demonstrar uma solução possível para transformar relatórios de vendas e compras em informação gerencial acessível. A validação real com usuários ainda é necessária para confirmar clareza, utilidade e próximos refinamentos.

## Próximos passos

- Aplicar teste de usabilidade com participante representativo.
- Registrar feedback real e refino da solução.
- Complementar evidências permitidas da entrega.
- Avaliar evolução técnica futura com backend, banco de dados e processamento real de arquivos.
