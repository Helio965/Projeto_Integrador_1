# Checklist da Entrega - NEXO (PI I)

Este checklist usa status textual para deixar claro o estágio de cada artefato sem inventar evidências.

## Legenda de status

- **Documentado** - artefato estruturado e disponível no repositório.
- **Protótipo criado** - item demonstrável no protótipo navegável.
- **Roteiro preparado** - documento pronto para orientar execução real.
- **Estrutura criada para preenchimento real** - modelo criado, dependente de dados reais da equipe.
- **Pendente de evidência real da equipe** - depende de link, teste, gravação, envio oficial ou comprovação real.
