## Organização sugerida

As Issues representam histórias de usuário e tarefas principais do MVP. Elas podem ser vinculadas a um GitHub Project ou Trello com as colunas:

| Coluna | Finalidade |
|---|---|
| Sprint Backlog | Histórias selecionadas para a Sprint. |
| To Do | Tarefas ainda não iniciadas. |
| In Progress | Tarefas em andamento. |
| Review/Testing | Tarefas em revisão ou teste. |
| Done | Tarefas concluídas. |
