# Backlog

Esta pasta reúne a organização ágil do projeto.

## Arquivos

- `product_backlog.md` — Product Backlog priorizado (US01–US22, MoSCoW).
- `sprint_backlog.md` — histórias por Sprint e tarefas T01–T35 com pontos.
- `backlog_resumido.md` — resumo por épicos.
- `issues_backlog.md` — Issues do GitHub e relação com o backlog.
- `plano_labels_milestones.md` — labels e milestones sugeridos.
- `github_projects_manual_setup.md` — orientação de quadro (GitHub Projects/Trello).

## Finalidade

Evidenciar a organização ágil do trabalho: do backlog de produto às tarefas das sprints, com rastreabilidade no GitHub.
