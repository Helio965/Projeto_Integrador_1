# Problema, Visão e Plano

Esta pasta reúne a base conceitual e de gestão do Projeto Integrador I: o problema real, os objetivos, a visão da solução e o plano de projeto.

## Arquivos

- `definicao_problema_objetivos.md` — definição do problema e objetivos do projeto.
- `documento_de_visao.md` — visão do produto, escopo do MVP e indicadores.
- `plano_de_projeto.md` — organização, equipe, cronograma e gestão do projeto.
- `evidencias.md` — registro de evidências permitidas.
- `termo_de_anuencia_orientacao.md` — orientação de privacidade sobre o termo de anuência.

## Finalidade

Demonstrar a identificação de um problema real e o planejamento do projeto, base para as etapas seguintes de Design Thinking, requisitos e execução.
