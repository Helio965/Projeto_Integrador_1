# Referências — NEXO

Esta pasta guarda material de **referência técnica** que apoia o NEXO, mas que **não faz parte do escopo central da entrega de Projeto Integrador I**.

## O que deve entrar aqui

- Material técnico consultado como referência (arquitetura, modelagem avançada etc.).
- Artefatos de outras disciplinas ou etapas que sirvam apenas como consulta.

## Regra de identidade da entrega

A entrega atual é de **Projeto Integrador I**. Documentos de referência de outras disciplinas (por exemplo, materiais de fases mais avançadas de arquitetura) podem servir como consulta, mas **não** devem virar capa, rótulo ou escopo central desta entrega.

Materiais que não pertencem à entrega de PI I, mas que se deseja preservar como consulta, devem ficar em:

```
referencias/nao_usar_na_entrega_pi1/
```
